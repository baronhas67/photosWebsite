# photosWebsite

Welcome to the website. This is website where I upload some private photos of
minecraft gameplay. Sometimes it's of me and my cousin.

There is not js here. Only html and css are used. And the photos are private so
we own the photos,

also there will be updates to the web. I know that it looks very basic now. The images 
have a description at the bottom.

## What type of photos?

There will be all types of photos from multiplayer to single player but mainly not
all or random photos will be uploaded. Only the good ones with good lighting or the
ones which very memorable will be uploaded. There will be 99.99% minecraft photos.

### Can I submit my photos?

Unfortunately no. Because this is personal website you can't submit your photos. Either
you will have to have a relation with me or my cousin.
